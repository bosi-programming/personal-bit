## Theme context with a provider that lets you customize the theme

A theme provider that lets you customize the theme used by the components on the library;

### Component usage
```js
<ThemeProvider theme="light">
Children here
<ThemeProvider/>
```

### Using props to customize the theme

Modify the text to see it change live:
```js live
<ThemeProviderWithTheme theme="dark" />
```

Bellow you can see the basic theme. You can overwrite any value bellow passing it to the ThemeProvider theme prop.
